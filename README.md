# Softmax Demo

This project demonstrates how the **Softmax function** converts raw scores (logits) into probability distributions for multi-class classification.  
It includes visual explanations, numerical examples, and plots to understand how softmax behaves with different inputs.

## Contents
- Softmax function implementation
- Visualizing probabilities for different logits
- Temperature scaling demonstration
- Comparing Softmax with Sigmoid
- Use-case in multi-class classification

## Key Concepts
- Why Softmax outputs sum to 1
- How logits influence final probabilities
- Effect of increasing or decreasing temperature
- Softmax in models like Logistic Regression & Neural Networks

## Libraries Used
- NumPy  
- Matplotlib  
- Scikit-learn (optional)
